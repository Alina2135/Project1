Цей проєкт розроблено в рамках практичної роботи.  
Метою є створення зручного графічного калькулятора з базовими математичними операціями.

## Короткий опис

Калькулятор надає користувачеві простий та інтуїтивно зрозумілий графічний інтерфейс для виконання арифметичних дій: додавання, віднімання, множення та ділення. Програма реалізована з використанням бібліотеки **Tkinter**, має захист від ділення на нуль та покриття модульними тестами.

### Використані технології
- Python 3.9+
- Tkinter (вбудована бібліотека для GUI)
- pytest (для тестування)
